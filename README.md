This website presents data and code related to the paper 'Three decades of experimental evolution studies on intralocus sexual conflict' by Holman, Thyagarajan, Grieshop and Chippindale, which is under review in _Philosophical Transactions of the Royal Society_.

Please click [here](https://lukeholman.github.io/exp_evol_metaanalysis/) to view a reproducible HTML report detailing the analysis, generating using [workflowr](https://github.com/workflowr/workflowr).
